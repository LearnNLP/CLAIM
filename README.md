# Legal Judgment Prediction

## Usage

The framework of this project is constructed according to [pytorch-worker](https://github.com/haoxizhong/pytorch-worker), you can visit the project for more details about usage.

In the directory ``config/element``, there exists three different directories ``large,small`` representing two parts of the dataset. Besides, in each directory, there exists several config files representing different models:

* ``bert.config``: BERT presented by Google.
* ``cnn.config``: TextCNN.
* ``dpcnn.config``: DPCNN.
* ``gating.config``: Deep Gating Network.
* ``lstm.config``: LSTM.
* ``topjudge.config``: TopJudge.
